Twitter图库抓取

目标市场画像 为什么选Twitter/ X？ 全球月活4.5亿用户，每天产生超过5亿条推文和图片。这些图片天然带有用户真实场景和高互动标签，非常适合用来训练AI视觉模型、生成社交媒体素材，或做竞品视觉分析。个人开发者无需任何成本即可获取海量素材。 适合卖什么？ 定制AI头像生成服务（用爬到的图做风格迁移训练） 社交媒体图片素材包（按主题打包，例如“科技创业者办公室实拍”） 电商产品图背景替换服务（用真实街景图做背景） 从零到上线的每一步 Step 1：注册Twitter开发者账号 访问 developer.twitter.com，点击“Sign up”。 使用个人邮箱注册，推荐用Gmail或Outlook。 填写基本信息：姓名、电话（可选）、国家。 选择“Individual Developer”计划——免费，每月可发送500条推文、读取15000条推文数据。 提交申请表，简述用途，例如“build a photo gallery app”。 审核通常24小时内通过，通过后你会获得以下信息： API Key API Key Secret Bearer Token（推荐用这个） 注意： 如果你的IP在国内，可能需挂梯子，否则注册页面加载缓慢。 Step 2：配置Python环境 安装Python 3.9以上版本，在终端运行： pip install requests 创建一个Python脚本 fetch_twitter_media.py Step 3：编写代码拉取图片 复制以下代码到文件中（替换 YOUR_BEARER_TOKEN）： import requests import json bearer_token = "YOUR_BEARER_TOKEN" def get_tweet_media(tweet_id): url = f"https://api.twitter.com/2/tweets/{tweet_id}?expansions=attachments.media_keys&media.fields=url,preview_image_url" headers = {"Authorization": f"Bearer {bearer_token}"} response = requests.get(url, headers=headers) if response.status_code != 200: print(f"Error: {response.status_code}, {response.text}") return [] data = response.json() media_urls = [] if "includes" in data and "media" in data["includes"]: for m in data["includes"]["media"]: if "url" in m: media_urls.append(m["url"]) return media_urls if __name__ == "__main__": # 找一个包含图片的推文ID，例如某网红发的图 tweet_id = "1790045219106652160" # 示例ID，实际请改 urls = get_tweet_media(tweet_id) for url in urls: print(url) 运行：python fetch_twitter_media.py，你会在终端看到图片URL列表。 ...
