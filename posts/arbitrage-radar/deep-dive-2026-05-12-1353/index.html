视频搜索淘金

核心洞察：为什么这个钱别人还没赚到？#

Reddit用户抱怨：“I realized that it’s pretty much impossible to search for something on YouTube, as there are only 5 or 6 relevant videos when you’re looking for some stuff, and then it’s all completely unrelated content.” 这条帖子获得200+赞，评论里全是 +1。现有方案如Yandex视频搜索不精准，Google视频搜索也是半成品。这是典型的信息差：你不需要做AI，只需调用免费API做一个垂直搜索页——比如只搜"Python爬虫教程"或"React组件库"，然后挂Google AdSense就能赚广告费。

Day1-7执行清单#

Day1（30分钟）：打开console.cloud.google.com→用Gmail注册→创建新项目→搜索YouTube Data API v3→启用它→创建API密钥→复制密钥保存到txt。

Day2（30分钟）：打开youtube.com/feed/explore→手动记录你感兴趣的5个垂直领域（如"Linux教程"“吉他谱"“健康餐”）→每个领域搜索前10个视频→记下观看量、标题、时长。

Day3（30分钟）：打开code.visualstudio.com→装VS Code→新建文件夹→创建index.html→粘贴以下代码并保存（替换YOUR_API_KEY）：

<!DOCTYPE html>
<html>
<head><title>我的垂直视频搜索</title></head>
<body>
<input id="q" placeholder="搜索关键词">
<button onclick="search()">搜</button>
<div id="results"></div>
<script>
function search() {
  const q = document.getElementById('q').value;
  fetch(`https://www.googleapis.com/youtube/v3/search?part=snippet&q=${q}&key=YOUR_API_KEY&maxResults=20`)
    .then(r=>r.json())
    .then(d=>{
      let html='';
      d.items.forEach(i=>{
        html+=`<div><a href="https://youtube.com/watch?v=${i.id.videoId}">${i.snippet.title}</a><p>${i.snippet.description}</p></div>`;
      });
      document.getElementById('results').innerHTML=html;
    });
}
</script>
</body>
</html>

然后用浏览器打开index.html，输入关键词测试。

Day4（30分钟）：打开netlify.com→用GitHub登录→点击New site from Git→上传index.html→获得可公开访问的URL→复制URL。

Day5（30分钟）：打开google.com/adsense→注册AdSense→等审核。同时发帖到V2EX/红迪：“我做了个X（如Linux教程）垂直视频搜索，搜关键词比YouTube更准”。

Day6（30分钟）：打开zhihu.com→搜"视频搜索工具推荐”→在问题下回答你的URL+说明优势。

Day7（30分钟）：打开plausible.io（有免费版）→注册→埋追踪脚本到你的搜索页→看前7天UV数据。

工具清单#

  • Google Cloud Console：免费，YouTube API每月免费1万次调用
  • Netlify：免费部署静态页
  • Google AdSense：审核通过后按点击付费
  • VS Code：免费
  • Plausible.io：免费版支持10000访问/月

收益预估#

  • 保守（每天100访问，点击率1%，CPC $0.1）：首月收益≈1000.010.1*30=30元
  • 中等（每天500访问，点击率2%，CPC $0.2）：首月收益≈5000.020.2*30=240元
  • 乐观（被大V转发，每天2000访问，点击率3%，CPC $0.3）：首月收益≈20000.030.3*30=540元

3个最容易踩的坑#

  1. API超额收费：YouTube API免费调用1万次/天，超过会扣费（$5/1000次）。解决办法：在Google Cloud Console设置每日配额上限为9000次。
  2. 搜索结果为空：某些关键词如"hantavirus"没有视频。Day1先手动验证该垂直领域有足够内容再开发。
  3. 展示侵权：不要抓取并展示完整视频，只展示标题和链接（已嵌入搜索API官方规则）。

  2. YouTube频道"The Net Ninja"免费API教程
