# Goose：零成本替代Claude Code的AI编程工具实测

## 工具卡片
- 名称：Goose
- 网址：github.com/block/goose
- 定价：完全免费（开源）
- 竞品：Claude Code（$200/月）、GitHub Copilot（$10/月）、Cursor（$20/月）
- 上手难度：3/5（需要基础命令行知识）
- 免费额度：无限使用，无任何限制

## 痛点与解决
你会写点JavaScript或Python，但老板让你一周改三个bug，而你完全没有头绪。Claude Code要200美元一个月，你连试都不想试。Goose就是答案——免费、开源、直接在终端里工作，像Claude Code一样能读代码、改代码、跑测试，但一分钱不花。

## 实测记录：同一个任务在Goose vs 手写
**任务**：用Python写一个函数，读取CSV文件并计算每列平均值。

**手写时间**：15分钟（找库、查语法、调试）
**Goose用时**：3分钟（写提示、看结果、微调）

### 操作步骤（从安装到出结果）：
1. 打开终端，执行：`npm install -g @block/goose`（需要Node.js 16+，没有的话先装Node.js）
2. 进入项目文件夹：`cd my-project`
3. 创建或打开一个空文件：`touch analyze.py`
4. 运行Goose：`goose code`
5. 输入提示（英文或中文）："Write a Python function to read a CSV file named data.csv and print the average of each numeric column"
6. Goose生成代码到analyze.py
7. 检查代码，如果有安装缺失（如pandas），看它自动提示，手动安装后再跑
8. 执行：`python analyze.py`，看控制台输出

### 进阶技巧：3个高效用法
1. **多文件修改**：先告诉Goose项目结构，再下命令"在app.js末尾添加一个路由处理GET /user"，它自动读取上下文，不改其他地方。
2. **修复bug**：运行程序报错后，把错误信息粘贴给Goose，说"这个报错怎么修"，它会解析错误并给出修改建议。
3. **文档生成**：对已有代码说"给这段代码写docstring"，它直接插入注释，节省半小时。

## 优缺点清单
### 优点
- 完全免费，无限次数
- 开源，社区活跃，不会突然收费
- 支持中文提示，对非英语开发者友好
- 直接在终端工作，不干扰编辑器

### 缺点
- 速度比商业产品慢
- 没有集成IDE，需要手动把代码复制到编辑器
- 对复杂项目有时理解不全，需要多次提示
- 不支持API集成（不能做批量任务）

### 替代方案
如果Goose不能满足，可以尝试以下免费选项：
- **Tabi**（tabi.sh）：免费AI编程，有网页版，无需命令行
- **Continue.dev**（continue.dev）：开源IDE插件，免费，需配合自己的API key
- **AiCoder**（github.com/aicoder）：另一个开源终端AI助手，功能相似

## 谁适合用/谁不适合用
**适合人群**：
- 会一点命令行（cd、npm install）的初级程序员
- 想做副业但不想花钱买工具的人
- 需要快速写小脚本或改bug的全职打工人

**不适合人群**：

- 需要实时代码补全（像Copilot那样边写边提示）的人
- 项目依赖复杂、需要AI理解大型框架的人
