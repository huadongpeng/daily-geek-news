## 效率痛点

每天花15分钟手动切换不同的AI模型API（Claude、GPT、Gemini）——修改参数、复制密钥、维护不同客户端库。一个月下来，这种重复操作消耗掉5个小时，相当于白白损失了500元（按100元/小时外包行情估算）。

## 自动化方案

### 工具链架构
- **Novastack**：统一路由层，通过配置文件管理所有模型。
- **Python 3.8+**：运行环境。

### 安装到首次运行

1. 安装Python 3.8+（如果已装则跳过）。
2. 打开终端，执行：
   ```bash
   pip install novastack
   ```
3. 在项目目录创建文件`router.py`：
   ```python
   from novastack import Router

   # 配置密钥（从环境变量读取更安全）
   router = Router({
       'openai': {'api_key': 'sk-xxx'},
       'anthropic': {'api_key': 'sk-ant-xxx'}
   })

   # 定义任务
   task = {'prompt': '解释量子纠缠', 'model': 'gpt-4'}
   
   # 自动路由到GPT-4
   result = router.route(task)
   print(result['content'])
   ```
4. 运行：`python router.py`。

### 核心配置代码
```python
# router.py - 完整示例
from novastack import Router

router = Router({
    'openai': {'api_key': os.environ['OPENAI_KEY']},  # 从环境变量加载
    'anthropic': {'api_key': os.environ['ANTHROPIC_KEY']}
})

# 根据模型名动态选择
task = {
    'prompt': '写一封求职信',
    'model': 'claude-3-opus-20240229'  # 换这里就能切换模型
}

response = router.route(task)
print(response['content'])
```

## 效果量化

- **优化前**：每次调用需手动编写请求代码+处理API错误。按每天调用10次，每次1.5分钟算：1.5分钟 × 10次/天 = 15分钟/天 = 5小时/月。
- **优化后**：一次配置，每次调用只需改`model`字段，30秒/次：0.5分钟 × 10次/天 = 5分钟/天 = 1.67小时/月。
- **节省**：3.33小时/月，如果用来接外包（按100元/小时），价值333元/月。

## 进阶技巧

1. **负载均衡**：设置`weight`参数按比例分配请求：`{'gpt-4': 0.7, 'claude-3': 0.3}`，避免单个模型超限。
2. **自动重试**：启用`retry=True`，当模型返回速率限制错误时自动换另一个模型重试，保证任务不中断。
3. **回滚策略**：在配置中加`fallback`字段，当首选模型失败时切换到低成本模型（如`gpt-3.5-turbo`）。

## 常见排障

1. **导入失败**: `ModuleNotFoundError: No module named 'novastack'`
   - 解决：确认pip安装成功，或尝试`python -m pip install --upgrade novastack`。
2. **模型名不识别**: 使用`claude-3-5-haiku`但报错
   - 解决：去Novastack官方文档查最新模型列表，注意版本号格式（如`claude-3-5-haiku-20241022`）。
3. **环境变量未加载**: 密钥为空报401
   - 解决：检查终端是否设置变量，或在代码里硬编码（仅测试用）。
