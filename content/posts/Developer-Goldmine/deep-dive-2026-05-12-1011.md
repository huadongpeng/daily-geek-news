## 效率痛点

你每天花30分钟回复类似的问题（发票过期、保险证书快到期），一个月就是 30分×20天=10小时。这些时间本来可以用来接一个 500 元的小外包项目。重复劳动是收入的天敌。

## 自动化/提效方案

### 工具链架构

- **触发器**：Gmail 收件箱（新邮件包含“保险证书/发票”）
- **处理节点**：n8n 内置的 Extract Data（提取附件、日期）
- **分支判断**：IF 证书到期 < 30 天 → 发提醒，否则归档
- **输出**：发送通知到 Telegram / Slack，同时更新 Google Sheets

### 从安装到首次运行

**第一步：注册 n8n 云端（免费）**

1. 打开 [n8n.io](https://n8n.io)，点击右上角“Get started free”
2. 用邮箱或 GitHub 账户注册
3. 登录后你会看到一个空的 workflow 画布

**第二步：创建你的第一个流程**

1. 点击“New Workflow”，命名为“Vendor Compliance Check”
2. 在右侧节点库搜索“Email Trigger (IMAP)”，拖到画布上
3. 配置 Gmail IMAP：填你的邮箱、密码（或App专用密码）
   - 设置筛选条件：subject contains "certificate" or "insurance"
4. 拖拽“IF”节点，连接到前一个节点
   - 条件：{{ $json["body"].expiry_date }} < {{ Date.now() + 30*24*60*60*1000 }}
5. 拖拽“Google Sheets”节点到 True 分支
   - 填你的 Sheet ID，列名：姓名、项目、到期日、状态
6. 拖拽“Telegram”节点到 False 分支
   - 输入 Bot Token、Chat ID，消息内容："{{ $json["sender"] }} 证书正常，无需关注"

**第三步：保存并激活**

1. 点击右上角“Save”，然后点“Activate”
2. 发送一封测试邮件，看流程是否跑通

### 核心配置代码（JSON 示例）

```json
{
  "nodes": [
    {
      "name": "Email Trigger",
      "type": "n8n-nodes-base.emailImap",
      "parameters": {
        "criteria": {
          "subject": "insurance|certificate"
        }
      }
    },
    {
      "name": "IF",
      "type": "n8n-nodes-base.if",
      "parameters": {
        "conditions": {
          "number": [
            {
              "value1": "={{ $json["expiry"] }}",
              "operation": "lessThan",
              "value2": "={{ Date.now() + 30*24*60*60*1000 }}"
            }
          ]
        }
      }
    }
  ]
}
```

### 进阶技巧

1. **并行处理**：在 IF 节点后加一个“Merge”节点，将 True 和 False 分支合并，然后统一更新一个状态看板。
2. **使用 Webhook**：不依赖轮询，用 n8n 的 Webhook 节点代替 IMAP，实时收到邮件。
3. **循环检查**：在 IF 节点前插入“Split In Batches”节点，一次处理多封邮件。

### 常见排障

1. **问题**：IMAP 连接失败。
   - **解决**：在 Gmail 安全设置中开启“低安全性应用访问”，或生成 App Password（Google 账号→安全→App Passwords）。
2. **问题**：n8n 免费版限制每月 2k 次执行。
   - **解决**：自建 Docker 容器（免费），或升级到企业版（按需付费）。
3. **问题**：提取日期失败。
   - **解决**：在 n8n 中手动添加“Extract from File”节点，解析 PDF/图片中的文本。
